# 常见故障速查表

## 光猫指示灯含义
| 指示灯 | 正常状态 | 异常含义 |
|--------|----------|----------|
| POWER | 常亮绿色 | 不亮=电源问题 |
| PON/注册 | 常亮绿色 | 闪烁=注册中，不亮=未注册 |
| LOS | 不亮 | 亮红色=光信号丢失 |
| LAN | 常亮/闪烁 | 不亮=网线未连接 |
| PHONE | 常亮 | 不亮=语音未注册 |

## 错误码速查
| 错误码 | 含义 | 处理方式 |
|--------|------|----------|
| 691 | 账号密码错误/被占用 | 验证账号密码、检查是否被盗用 |
| 678 | 服务器无响应 | 检查线路、报修 |
| 651 | 调制解调器错误 | 重启光猫、检查驱动 |
| 734 | PPP连接控制协议终止 | 重建拨号连接 |
| 735 | 服务器拒绝请求地址 | 检查VLAN配置 |
| 718 | 服务器无响应超时 | 检查线路质量 |

## 光功率参考值
- 范围: -8dBm ~ -27dBm
- 优良: -8 ~ -20dBm
- 正常: -20 ~ -25dBm
- 偏弱: -25 ~ -27dBm（可能出现丢包）
- 异常: >-27dBm（需检修）
