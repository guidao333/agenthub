## 安全边界

- 本地工作台只监听 `127.0.0.1`。
- 本地 API 使用随机 token。
- 未携带 token 的 API 请求返回 `403`。
- 本地配置只保存客户内网参数和客户私有规则。
- 官方能力核心逻辑、提示词和训练成果不以明文包形式下发。
- 本地工作台不提供能力包导出、插件目录读取、安装目录打包或任意命令执行接口。
